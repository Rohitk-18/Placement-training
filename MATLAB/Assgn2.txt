clc, clearvars;

%1. Write a script that takes a number as input and prints whether it is positive, negative, or zero.

num = input('Enter a number: ');
if num > 0
    disp('The number is positive');
elseif num < 0
    disp('The number is negative');
else
    disp('The number is zero');
end


%2. Using a for loop, calculate the sum of the first n natural numbers, where n is given by the user.

n = input('Enter a positive integer: ');
sum = 0;
for i = 1:n
    sum = sum + i;
end
fprintf('Sum of first %d natural numbers is %d\n', n, sum);
